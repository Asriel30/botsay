# botsay

Like cowsay, but with a small ASCII bot from [go-asciibot](https://github.com/mattes/go-asciibot).

### Examples

Example output of `botsay -- $(fortune)`:

```
       .-.           .------------------------------------------.
    ._(u u)_.        | Nothing increases your golf score like   |
      (_n_)       --<| witnesses.                               |
    .=[::+]=.        '------------------------------------------'
  ]=' [___] '=[                                                  
      /| |\                                                      
     (0) (0)
```

```

   -)O O(-         | )_( .--. |- -| '--' |_^_| o==|ooo|==o      |
    \_0_/       --<| |___| // \\ _\\ //_ .===./` .--. /.- -.    |
 .==|>o<|==:=L     | \ '--' "\_n_/" ,"|+ |". _\|+__|/_ ( )      |
 '=c|___|          | __) (__ _._._ .--. -)ooo(- '--' \_=_/      |
    ]| |[          | (m9\:::/\ /___\6 . /___\ . . ..:::::::. .  |
   [_| |_]         '--------------------------------------------'
```
